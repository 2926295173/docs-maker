:::warning 其他注意事项
1. 关于 2025 年多个刷课问题公示 ： [点击查看](./issues/2025.md)
1. 在使用之前，请关闭其他同类型脚本，避免发生冲突 ！如果页面出现多个 OCS 脚本，请参考下方的 [删除脚本教程](#删除脚本教程) ，确保只有一个脚本
1. 在安装页面点 `安装/更新` 按钮后，页面会 `自动关闭` ，这是 `正常现象`
1. 如果是超星没出现，但是智慧树等其他平台出现了悬浮窗，查看教程 [#为什么超星不显示脚本但其他网课显示](/docs/other/FQA#chaoxing-not-show)
